# 📋 Журнал изменений — GeoDistanceTool CLI

Все заметные изменения в этом проекте документируются в этом файле.

---

## [1.0.1] — 2026-04-10

### 🛠 Исправления

- **CLI**: Исправлено мгновенное закрытие окна при запуске `.exe` двойным кликом  
  → Теперь программа корректно определяет, запущена ли она в собственном окне консоли  
  → Пауза «Нажмите любую клавишу» показывается только при запуске двойным кликом на Windows  
  → При запуске из терминала (CMD/PowerShell) или с перенаправлением потоков (`>`, `|`) пауза не мешает работе  
  → Добавлена обработка `Ctrl+C` с кодом выхода `130` (стандарт SIGINT)

### 🔧 Технические детали

- Добавлены WinAPI-импорты (`GetConsoleWindow`, `GetWindowThreadProcessId`) для определения владельца окна
- Реализован метод `IsRunningInOwnConsoleWindow()` для точной детекции контекста запуска
- Улучшена логика `Program.cs`: проверка `Console.IsInputRedirected` / `IsOutputRedirected` перед `Console.ReadKey()`

### 👥 Для кого это важно

| Пользователь | Что изменилось |
|--------------|----------------|
| 🖱️ Обычный пользователь (Windows) | Окно больше не закрывается сразу — можно скопировать результат |
| ⚙️ Продвинутый пользователь / скрипты | При редиректе (`app.exe > log.txt`) пауза не блокирует автоматизацию |
| 🐧 Пользователь Linux/macOS | Поведение не изменилось (пауза только на Windows) |

## [1.0.0] — 2026-04-09

### 🎉 Первый стабильный релиз

#### ✨ Возможности
- 🧭 Расчёт расстояния по формуле гаверсинусов (учёт кривизны Земли)
- ⌨️ Два режима работы: через аргументы командной строки и интерактивный ввод
- ✅ Валидация координат: широта (−90…90), долгота (−180…180)
- 🎨 Красивый вывод с цветами и эмодзи через `Spectre.Console`
- 🪟 Установка в один клик через `install.ps1` (добавление в `PATH`, ярлык на рабочем столе)
- 🌐 Кроссплатформенность на .NET 10

#### 📦 Артефакты
- `GeoDistanceTool.exe` — исполняемый файл
- `install.ps1` / `uninstall.ps1` — скрипты установки и удаления
- Исходный код и тесты в репозитории

#### 🧪 Тестирование
- Покрытие на xUnit: расчёты, валидация, CLI-интеграция, негативные сценарии
